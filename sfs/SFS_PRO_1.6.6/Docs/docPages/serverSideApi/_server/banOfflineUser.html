_server.banOfflineUser()

Availability:

SmartFoxServer PRO 1.6.3

Usage:

_server.banOfflineUser(userName)

Description:

Bans an off-line user from the server.

Properties:

userName   the name of the user to ban

Returns:

nothing.

Example:

/*
* Bans an offline user
*/

_server.banOfflineUser("Lapo")

See also:

kickUser, banUser, removeBanishment
